ragas_embedding

 ragas_embedding (provider:str, model:str, client:Any)

*Factory function to create an embedding instance based on the provider.

Args: provider (str): The name of the embedding provider (e.g., “openai”). model (str): The model name to use for embeddings. **kwargs: Additional arguments for the provider’s client.

Returns: BaseEmbedding: An instance of the specified embedding provider.*
